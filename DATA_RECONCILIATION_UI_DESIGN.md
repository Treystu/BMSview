## Color Coding & Visual Hierarchy

### Summary Cards
- **Blue** (Total DL Sources): Informational, neutral
- **Green** (Matched): Positive, healthy status
- **Yellow** (Orphaned): Warning, needs attention
- **Purple** (Orphaned Records): Data waste indicator

### Tables
- **Yellow Border/Background** (Orphaned Section): Warning state
- **Green Header** (Matched Section): Success state
- **Blue Checkboxes**: Selected items for merge
- **Gray Hover**: Interactive row hover effect

### Buttons
- **Green** "âž• Adopt System": Primary action for orphans
- **Blue** "ðŸ”„ Refresh": Secondary action
- **Secondary** "ðŸ”€ Merge X Systems": Important bulk action
- **Blue** "âœï¸ Edit": Standard edit action
- **Blue** "ðŸ“ Use Current Location": Geolocation trigger

## Responsive Behavior

### Desktop (>1024px)
- Summary cards: 4 columns
- Tables: Full width, all columns visible
- Modal: Centered, 512px max width

### Tablet (768px - 1024px)
- Summary cards: 2 columns, 2 rows
- Tables: Horizontal scroll for overflow
- Modal: Responsive padding

### Mobile (<768px)
- Summary cards: 2 columns, 2 rows (smaller font)
- Tables: Vertical card layout (future enhancement)
- Modal: Full screen overlay

## User Feedback Patterns

### Success: System Created
- Modal closes automatically
- Dashboard refreshes
- DL-# moves from orphaned to matched
- Summary cards update

### Success: Systems Merged
- Selections clear
- Dashboard refreshes
- Primary system shows all DL-#s
- Other systems removed from table

### Error: System Creation Failed
- Alert dialog with error message
- Modal stays open (user can retry)
- Form data preserved

## Accessibility Features

### Screen Reader Support
- All form labels properly associated
- Table headers have semantic markup
- Button aria-labels for icon-only actions
- Modal has `role="dialog"` and `aria-modal="true"`

### Keyboard Navigation
- Tab order follows visual flow
- Enter key submits forms
- Escape key closes modal
- Checkbox keyboard selection

### Focus Management
- Modal traps focus
- First input focused on modal open
- Focus returns to trigger after close
